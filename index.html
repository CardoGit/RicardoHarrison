Diabetic Retinopathy Detection using Computer Vision

DeepLearning RetinalImaging AIForHealth ComputerVision DiabeticRetinopathy

This project aims to enable early detection of diabetic retinopathy by applying deep learning and computer vision techniques to retinal images. Using a ResNet-based convolutional neural network, the system is trained to classify the presence and severity of diabetic retinopathy with high accuracy.

By leveraging advanced image processing and transfer learning, this solution helps healthcare providers identify at-risk patients early, improving treatment outcomes and expanding access to reliable diagnostics, particularly in low-resource settings.

AI-Based Blood Pressure and Vital Signs Monitoring System

NonInvasiveBP Telemedicine RemotePatientMonitoring VitalSigns PPG

This project is a non-invasive, AI-powered system designed to estimate blood pressure and monitor key vital signs in real time. Using a MAX30102 optical sensor, the system collects IR and RED PPG signals to extract features such as standard deviation, delta, pulse amplitude, heart rate variability, and heart rate. These features are processed by a trained machine learning model, achieving blood pressure estimation within ±4 mmHg accuracy.

In addition to blood pressure, the system measures heart rate, SpO₂, and body temperature. A custom mobile app displays the readings, allows users to log in, and enables secure messaging with their doctor if needed. All patient data is also available through a connected web dashboard, providing healthcare professionals with convenient access to up-to-date patient information.

Grip Strength Monitor

GripStrength PressureTransducer EmbeddedSystems

This project involves creating a wearable device that measures grip strength using a pressure transducer paired with a squeezable rubber bung. The device is worn around the bicep, allowing for convenient, real-time monitoring of grip force.

Designed for use in physical therapy, athletic training, and rehabilitation, the system offers precise measurements to help track and improve muscle performance outside of clinical settings. By integrating sensor technology with custom electronics, this solution provides a practical and portable method for assessing muscular strength.
